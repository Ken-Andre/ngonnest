## Ordre de Lecture Recommandé

Pour un nouveau développeur ou IA :

1. **AI_RULES.md** - Comprendre la philosophie et les contraintes
2. **requirements.md** - Connaître les requirements V1 prioritaires
3. **tasks_v1.md** - Identifier la tâche à exécuter
4. **AGENT_PROMPT.md** - Comprendre le workflow de travail
5. **design.md** - Voir l'architecture technique
6. **AGENTS.md** - Checklist quotidienne avant commit

## Phases du Projet

### V1 (MVP OFFLINE-ONLY) - 12-16 semaines
- 100% offline, store-ready
- Focus : stabilité, UX claire, performance

### V2 (POST-MVP) - 8-10 semaines après V1
- Graphs budget, export PDF, IA on-device
- Monétisation (premium)

### V3+ (LONG-TERME) - Selon feedback utilisateurs
- Cloud sync (opt-in)
- Multi-device, partage famille
- Mode entreprise (hôtels/restos)

## Critère de Qualité
